# cmdNapis
Program rysuje w konsoli napis który podaliśmy, lecz większymi literami zapisanymi znakami specjalnymi.  
![Obrazek pokazujący program #1]( https://i.imgur.com/dvZi3HF.png)
![Obrazek pokazujący program #2]( https://i.imgur.com/i6Qztu1.png)

### Działanie programu
Program prosi użytkownika o podanie wyrazu, po czym rysuje go w konsoli za pomocą „#” oraz w większej formie. Najciekawsze jednak jest to jak program rozpoznaje jak namalować daną literę, każdy znak jest podzielony na 5 poziomych elementów. Program posiada tylko bazę elementów gdzie jest zapisane tylko 10 elementów z których potem są składane znaki, lepiej widać to na 2 obrazku, gdzie jest to rozpisane w tabeli, widać wtedy z jakich elementów składa się dany znak.
